# The Inspire Space 🏢
### iad Real Estate · Orlando, Florida

Room booking platform for The Inspire Space co-working office.

## Spaces
- 📚 **Training Room** — 20 seats, requires admin approval
- 🎙️ **Podcast Studio** — 4 seats, instant booking
- 🗼 **Paris Meeting Room** — 8 seats, instant booking
